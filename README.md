# 📊 Portfolio360

## این اپ چیست؟

Portfolio360 ابزاری برای دو دسته تصمیم مالی است:

1. **تحلیل استراتژی‌های آپشن** (Covered Call، Protective Put، Long Strangle و بیشتر) — با ورودی‌های واقعی (قیمت خرید، استرایک، پرمیوم، سررسید)، اپ نقطه‌ی سر‌به‌سر، حداکثر سود/زیان، بازده سالانه‌شده، و یک **نمودار سود/زیان تعاملی** رسم می‌کند؛ و مهم‌تر، بر اساس بازده موردانتظار و نرخ بدون ریسکی که کاربر وارد می‌کند، یک **استدلال شفاف** می‌دهد که آیا این معامله به‌صرفه است یا نه.
2. **بهینه‌سازی وزن پرتفوی** روی چند دارایی (سهام، ETF، کریپتو، کالا) با چهار روش مکمل: **Monte Carlo** (جست‌وجوی وزن بهینه با هزاران سناریوی شبیه‌سازی‌شده)، **CVaR** (کمینه‌سازی ریسک دنباله)، **Stress Test** (عملکرد در بدترین سناریوهای تاریخی)، و **Barbell** (ترکیب امن+پرریسک به سبک نسیم طالب).

داده‌های زنده (قیمت لحظه‌ای و زنجیره آپشن) از [Derive.xyz](https://derive.xyz) خوانده می‌شود.

## ویژگی‌های کلیدی

- 🔗 **زنجیره آپشن زنده** با گریک‌ها (Δ/Γ/V/Θ) برای انتخاب قرارداد واقعی
- 📈 **نمودار سود/زیان تعاملی** (MPAndroidChart) برای هر استراتژی، با نقاط سر‌به‌سر و قیمت لحظه‌ای مشخص‌شده
- 🧠 **استدلال سرمایه‌گذاری خودکار** — مقایسه‌ی بازده/هزینه‌ی هر استراتژی با بازده موردانتظار و نرخ بدون ریسک کاربر
- 🧮 **بهینه‌سازی پرتفوی** با ۴ روش (Monte Carlo، CVaR، Stress Test، Barbell) و نمایش وزن‌ها به‌صورت درصد و مبلغ
- 🔐 **ورود با ایمیل/رمز یا Google** (Firebase Authentication) — هر حساب مستقل و همگام روی همه‌ی دستگاه‌ها
- 🎁 **۱۰ روز دوره آزمایشی رایگان** برای حساب‌های تازه‌ساز (بر اساس تاریخ ساخت حساب در سرور)
- 💳 **پرداخت اشتراک با کارت بانکی یا کریپتو** (USDT روی شبکه‌ی Arbitrum/BEP20) و تأیید از طریق تلگرام
- 🛠 **پنل مدیریت داخل اپ** برای فعال‌سازی دستی اشتراک، بدون نیاز به سرور یا دخالت شخص ثالث
- 👤 **پروفایل کاربری با بک‌آپ ابری** (عکس + نام روی Firestore، بدون نیاز به سرویس پولی Storage)
- 💬 **پرسش‌های متداول + ارسال مستقیم انتقاد به تلگرام** از داخل اپ
- 🗺 **راهنمای خرید واقعی دارایی‌ها** (سهام آمریکایی، آپشن، کالا) با معرفی پلتفرم و هشدار نیاز به VPN
- 🌗 تم روشن/تاریک خودکار، هماهنگ با تنظیمات گوشی

## معماری فنی

| بخش | فناوری |
|---|---|
| زبان و پلتفرم | Java، Android SDK (minSdk 24، targetSdk 34) |
| احراز هویت و دیتابیس | Firebase Authentication + Cloud Firestore |
| شبکه | OkHttp + Derive.xyz Public API |
| نمودار | MPAndroidChart |
| اعلان/ارتباط | Telegram Bot API (مستقیم از کلاینت، بدون سرور واسط) |
| CI/CD | GitHub Actions (بیلد خودکار APK دیباگ در هر push) |

### ساختار پکیج‌ها

```
com.mamad.portfolio360/
├── wizard/       # مسیر انتخاب استراتژی، زنجیره آپشن، نتیجه/نمودار
├── portfolio/     # ورودی دارایی و موتور بهینه‌سازی پرتفو
├── premium/       # هاب اشتراک، اطلاعات پرداخت، مدیریت وضعیت اشتراک
├── admin/         # پنل مدیریت و نگاشت ایمیل↔uid
├── profile/       # پروفایل کاربر (عکس/نام/خروج از حساب)
├── support/       # سوالات متداول + فرم انتقاد
├── calc/          # موتورهای محاسباتی خالص (Payoff، CVaR، Monte Carlo، Black-Scholes)
└── network/       # کلاینت API قیمت/زنجیره آپشن
```

## چطور نسخه‌ی تست را نصب کنم؟

1. وارد تب **Actions** این ریپو شوید.
2. آخرین اجرای موفق (✅) ورک‌فلوی `Android Build` را باز کنید.
3. پایین صفحه، در بخش **Artifacts**، فایل `portfolio360-debug-apk` را دانلود کنید.
4. فایل zip را باز کنید و `app-debug.apk` را روی گوشی اندروید نصب (sideload) کنید — نیاز به فعال‌کردن «نصب از منابع ناشناس» دارید.

> این یک بیلد دیباگ برای تست است، نه نسخه‌ی نهایی منتشرشده در فروشگاه اپلیکیشن.
