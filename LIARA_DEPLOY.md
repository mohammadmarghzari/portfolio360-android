# راه‌اندازی ربات تأیید خودکار اشتراک روی لیارا

این راهنما قدم‌به‌قدم می‌گوید چطور سرور تأیید خودکار اشتراک (پوشه‌ی `server/`) را روی
[لیارا](https://liara.ir) مستقر کنی. بعد از این کار، دیگر نیازی به تأیید دستی نیست:
کاربر رسید را برای ربات می‌فرستد، تو توی تلگرام روی پلن می‌زنی، و اشتراکش خودکار فعال می‌شود.

## پیش‌نیازها

- یک حساب لیارا (در `liara.ir` ثبت‌نام کن و کمی شارژ کن — هزینه‌ی ماهانه‌ی یک اپ کوچک Node ناچیز است).
- نصب Node.js روی کامپیوترت (از `nodejs.org` نسخه‌ی LTS).
- سه مقدار حساس که باید آماده داشته باشی:
  1. **BOT_TOKEN** — توکن رباتت که از BotFather گرفتی.
  2. **ADMIN_CHAT_ID** — آیدی عددی چت خودت (از ربات `@userinfobot`).
  3. **FIREBASE_SERVICE_ACCOUNT** — کل محتوای فایل service-account فایربیس (همان فایل JSON کلید خصوصی).

## مرحله ۱: نصب و ورود به ابزار لیارا

روی کامپیوترت ترمینال را باز کن و بزن:

```bash
npm install -g @liara/cli
liara login
```

## مرحله ۲: ساخت اپ در لیارا

- وارد داشبورد لیارا شو → «برنامه‌ها» → «ایجاد برنامه».
- پلتفرم را **Node.js** انتخاب کن و یک نام بگذار (مثلاً `portfolio360-bot`).

## مرحله ۳: تنظیم متغیرهای محیطی (مهم — اینجا کلیدها امن می‌مانند)

در داشبورد همان اپ، بخش **«متغیرهای محیطی»** (Environment Variables) این سه مورد را اضافه کن:

| نام | مقدار |
|---|---|
| `BOT_TOKEN` | توکن ربات از BotFather |
| `ADMIN_CHAT_ID` | آیدی عددی چت خودت |
| `FIREBASE_SERVICE_ACCOUNT` | کل محتوای فایل JSON کلید فایربیس را کپی و اینجا پیست کن |

> کلیدها فقط اینجا (روی سرور) می‌مانند و هیچ‌وقت داخل کد یا اپ اندروید نیستند.

## مرحله ۴: استقرار

در ترمینال، وارد پوشه‌ی `server` این پروژه شو و بزن:

```bash
cd server
liara deploy
```

نام اپی که ساختی را انتخاب کن. بعد از چند دقیقه، سرور بالا می‌آید و **دائمی روشن می‌ماند**.
ربات از همان لحظه شروع به کار می‌کند.

## مرحله ۵: تست

- از یک اکانت تلگرام دیگر (نه اکانت مدیر)، ربات را با لینکی که اپ می‌سازد باز کن.
- یک عکس (به‌عنوان رسید آزمایشی) برای ربات بفرست.
- باید در چت مدیر، عکس با دکمه‌های «۱ ماهه / ۳ ماهه / ۶ ماهه / ۱ ساله» بیاید.
- روی یک پلن بزن → باید پیام «فعال شد ✅» بیاید و اشتراک آن کاربر در اپ فعال شود.

## بعدش

وقتی سرور بالا آمد و تست شد، به من بگو تا **دکمه‌ی «ارسال رسید» اپ را به ربات وصل کنم**
(الان به‌صورت موقت مستقیم به چت مدیر می‌رود). بعد از آن، کل چرخه‌ی «رسید → تأیید → فعال‌سازی»
کاملاً خودکار می‌شود.

## هزینه‌ها

- فقط هزینه‌ی ماهانه‌ی همان یک اپ کوچک Node در لیارا (به‌صورت ریالی، معمولاً بسیار کم).
- بدون هیچ هزینه‌ی دیگری؛ فایربیس در سطح رایگان (Spark) برای این حجم کافی است.
