# راه‌اندازی ربات تأیید خودکار اشتراک روی سرور اوبونتوی لیارا

تو یک **سرور ابری (VM) اوبونتو** روی لیارا ساختی. این راهنما می‌گوید چطور ربات تأیید
خودکار اشتراک (پوشه‌ی `server/`) را روی همان سرور اجرا کنی تا دائمی روشن بماند.

بعد از این کار: کاربر رسید را برای ربات می‌فرستد، تو توی تلگرام روی پلن می‌زنی، و
اشتراکش خودکار فعال می‌شود.

---

## چیزهایی که لازم داری

1. **IP سرور و رمز root** — در داشبورد لیارا، صفحه‌ی همین سرور (`port-360`)، بخش
   «تنظیمات» یا «نمای کلی»، آدرس IP و رمز عبور (یا کلید SSH) را پیدا کن.
2. **توکن ربات** از BotFather.
3. **فایل کلید فایربیس** (همان `service-account.json` که قبلاً داشتی).

---

## مرحله ۱: اتصال به سرور با SSH

روی کامپیوترت (ویندوز: PowerShell، مک/لینوکس: Terminal) بزن — به‌جای `IP_SERVER` آدرس
سرورت را بگذار:

```bash
ssh root@IP_SERVER
```

رمز عبور را وارد کن (موقع تایپ رمز چیزی نمایش داده نمی‌شود، طبیعی است).

## مرحله ۲: نصب پیش‌نیازها (یک‌بار)

بعد از اتصال، این بلوک را کامل کپی و پیست کن و Enter بزن:

```bash
apt update
apt install -y nodejs npm git
npm install -g pm2
```

## مرحله ۳: گرفتن کد

```bash
git clone https://github.com/mohammadmarghzari/portfolio360-android.git
cd portfolio360-android/server
npm install
```

## مرحله ۴: گذاشتن کلید فایربیس روی سرور

فایل کلید را می‌سازیم و محتوایش را پیست می‌کنیم:

```bash
nano service-account.json
```

حالا **کل محتوای فایل `service-account.json`** خودت را (که از فایربیس گرفتی) کپی کن و
داخل این پنجره پیست کن. بعد `Ctrl+X` بعد `Y` بعد `Enter` تا ذخیره شود.

## مرحله ۵: تنظیم مقادیر محرمانه

```bash
cp .env.example .env
nano .env
```

مقدارها را این‌طور کن (توکن رباتت را جایگزین کن؛ بقیه معمولاً درست‌اند):

```
BOT_TOKEN=توکن_واقعی_رباتت
ADMIN_CHAT_ID=840723814
FIREBASE_SERVICE_ACCOUNT=/root/portfolio360-android/server/service-account.json
```

`Ctrl+X` بعد `Y` بعد `Enter`.

## مرحله ۶: اجرای دائمی ربات

```bash
pm2 start index.js --name portfolio360-bot
pm2 save
pm2 startup
```

(دستور آخر یک خط دیگر به تو می‌دهد که باید همان را هم کپی و اجرا کنی تا ربات بعد از
ری‌استارت سرور هم خودکار روشن شود.)

برای دیدن وضعیت و لاگ‌ها:

```bash
pm2 logs portfolio360-bot
```

اگر پیام `Health server up. Bot polling started.` را دیدی، یعنی ربات کار می‌کند. ✅

## مرحله ۷: تست

- از یک اکانت تلگرام دیگر (نه اکانت مدیر)، ربات را باز کن و `/start` بزن، بعد یک عکس
  (به‌عنوان رسید آزمایشی) بفرست.
- باید در چت مدیر، عکس با دکمه‌های «۱ ماهه / ۳ ماهه / ۶ ماهه / ۱ ساله» بیاید.
- روی یک پلن بزن → پیام «فعال شد ✅» بیاید.

## به‌روزرسانی کد در آینده

هر وقت کد سرور را تغییر دادم، فقط این‌ها را روی سرور بزن:

```bash
cd portfolio360-android
git pull
cd server
npm install
pm2 restart portfolio360-bot
```

## بعدش

وقتی ربات بالا آمد و تست شد، به من بگو تا **دکمه‌ی «ارسال رسید» اپ را به ربات وصل کنم**
(الان موقتاً مستقیم به چت مدیر می‌رود). بعد از آن، کل چرخه کاملاً خودکار می‌شود.
